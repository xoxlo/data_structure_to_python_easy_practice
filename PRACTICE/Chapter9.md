# CHAPTER 9 탐색트리
- 탐색트리 : 탐색을 위한 트리 기반의 자료구조
- 이진탐색트리(BST) : 효율적인 탐색을 위한 이진트리 기반의 자료구조
- 이진탐색트리의 조건 -> 인쪽 서브트리 노드의 키값 < 루트 노드의 키값 < 오른쪽 서브트리 노드의 키값
- 모든 노드는 유일한 키를 갖는다

## 순차탐색
- 시간복잡도 O(n)

## 이진탐색
- 시간복잡도 O(log_2n)

## 해싱
- 시간복잡도 O(1)
